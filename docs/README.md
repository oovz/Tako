# Documentations

This directory collects the contributor-facing documentation for Tako.

## Start with the right guide

- **Core extension work**
  Read `ARCHITECTURE.md` first if you are touching the background runtime, side panel, options page, shared state, storage, or test infrastructure.

- **Site integration work**
  Read `CONTRIBUTING-SITE-INTEGRATION.md` first if you are adding a new supported site or maintaining an existing integration.

## Supporting references

- `MESSAGING.md` - focused reference for runtime message types, sender rules, and offscreen boundaries
- `TEMPLATE-MACROS.md` - filename and path-template macro reference used by download settings

## Recommended reading order

### For most contributors

1. `../README.md`
2. `ARCHITECTURE.md`
3. `MESSAGING.md`

### For site integration contributors

1. `../README.md`
2. `CONTRIBUTING-SITE-INTEGRATION.md`
3. `ARCHITECTURE.md`
4. `TEMPLATE-MACROS.md`
