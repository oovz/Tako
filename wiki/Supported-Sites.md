# Supported Sites

Tako supports manga, manhua, and manhwa sites through purpose-built integrations. Each site gets optimized handling for its page structure, image CDN, and authentication, so downloads are reliable and fast.

## Live integrations

| Site | Status | Notes |
|------|--------|-------|
| MangaDex | ✅ | The largest manga catalog; Tako uses your browser session seamlessly. |
| Pixiv Comic | ✅ | Authenticated session handling for Pixiv content. |
| Shonen Jump+ | ✅ | Direct chapter and series page support. |
| Manhuagui | ✅ | Chinese manhua with dedicated parser. |
| Comic Nettai | ✅ | Japanese manga with optimized extraction. |

## More sites coming

Want a new site? Open a [site request issue](https://github.com/oovz/Tako/issues/new?template=site_request.md) or contribute the integration yourself — see the [Site Integration Guide](Site-Integration-Guide).

[Install Tako from the Chrome Web Store](https://chromewebstore.google.com/detail/tako-manga-downloader/hlodmckfkmbenkknmailfekehgajpmbb).
